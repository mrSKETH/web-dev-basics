Введение в CSS

CSS (Cascading Style Sheets) - это язык стилей, который используется для описания внешнего вида документов, написанных на HTML. CSS позволяет контролировать макет, цвета, шрифты и другие аспекты визуального представления веб-страниц.

Важно!

CSS отделяет содержание (HTML) от представления (стили), что делает код более чистым и удобным для поддержки.

Что можно делать с помощью CSS?

  • Изменять цвета текста и фона
  • Настраивать шрифты и их размеры
  • Создавать сложные макеты страниц
  • Добавлять анимации и переходы
  • Создавать адаптивные дизайны для разных устройств
  • И многое другое!

Синтаксис и подключение CSS

CSS состоит из набора правил, каждое из которых содержит селектор и блок объявлений.

/* Это комментарий в CSS */
селектор {
    свойство: значение;
    другое-свойство: значение;
}

Способы подключения CSS

1. Встроенные стили (inline)

<div style="color: blue; font-size: 16px;">Текст с встроенными стилями</div>

2. Внутренние стили (internal)

<head>
    <style>
        body {
            background-color: #f0f0f0;
        }
        h1 {
            color: navy;
        }
    </style>
</head>

3. Внешние стили (external)

<head>
    <link rel="stylesheet" href="styles.css">
</head>

Рекомендация

Используйте внешние стили для больших проектов - это упрощает поддержку и позволяет переиспользовать стили на разных страницах.

Селекторы

Селекторы определяют, к каким элементам HTML будут применены стили.

Основные типы селекторов

1. Селектор по тегу

p {
    color: red;
}

2. Селектор по классу

.my-class {
    font-weight: bold;
}

3. Селектор по ID

#my-id {
    background-color: yellow;
}

4. Универсальный селектор

* {
    margin: 0;
    padding: 0;
}

Комбинированные селекторы

1. Группировка селекторов

h1, h2, h3 {
    font-family: Arial, sans-serif;
}

2. Вложенные селекторы

div p {
    color: blue;
}

3. Дочерние селекторы

div > p {
    font-weight: bold;
}

4. Соседние селекторы

h1 + p {
    margin-top: 0;
}

Псевдоклассы и псевдоэлементы

Псевдоклассы

a:hover {
    color: red;
}

li:first-child {
    font-weight: bold;
}

input:focus {
    border-color: blue;
}

Псевдоэлементы

p::first-line {
    font-weight: bold;
}

p::before {
    content: "→ ";
}

p::after {
    content: " ←";
}

Цвета и фон

CSS предоставляет различные способы задания цветов и фоновых изображений.

Способы задания цветов

1. Именованные цвета

color: red;
background-color: blue;
border-color: green;

2. HEX-значения

color: #ff0000;        /* Красный */
color: #00ff00;        /* Зеленый */
color: #0000ff;        /* Синий */
color: #ffffff;        /* Белый */
color: #000000;        /* Черный */
color: #336699;        /* Сине-серый */

3. RGB и RGBA

color: rgb(255, 0, 0);        /* Красный */
color: rgba(255, 0, 0, 0.5);  /* Красный с прозрачностью 50% */

4. HSL и HSLA

color: hsl(0, 100%, 50%);        /* Красный */
color: hsla(0, 100%, 50%, 0.5);  /* Красный с прозрачностью 50% */

Текст и шрифты

CSS предоставляет множество свойств для оформления текста.

Свойства шрифтов

    font-family: Arial, Helvetica, sans-serif;
    font-size: 16px;
    font-weight: normal; /* или bold, 100-900 */

    line-height: 1.5;
}

Свойства текста

p {
    color: #333;
    text-align: left; /* или center, right, justify */
    text-decoration: none; /* или underline, overline, line-through */
    text-transform: none; /* или uppercase, lowercase, capitalize */
    letter-spacing: 1px;
    word-spacing: 2px;
    text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
}

Позиционирование

Свойство position определяет, как элемент позиционируется на странице.

Типы позиционирования

.static {
    position: static; /* По умолчанию */
}

.relative {
    position: relative;
    top: 10px;
    left: 20px;
}

.absolute {
    position: absolute;
    top: 0;
    right: 0;
}

.fixed {
    position: fixed;
    bottom: 20px;
    right: 20px;
}

.sticky {
    position: sticky;
    top: 0;

Свойства z-index

Свойство z-index контролирует порядок наложения элементов друг на друга.

.element1 {
    position: relative;
    z-index: 1;
}

.element2 {
    position: relative;
    z-index: 2; /* Будет поверх element1 */
}

Flexbox

Flexbox - это модуль макета, который упрощает создание гибких и адаптивных макетов.

Основные понятия Flexbox

  • Flex container - родительский элемент с display: flex
  • Flex items - дочерние элементы контейнера
  • Main axis - основная ось (горизонтальная по умолчанию)
  • Cross axis - поперечная ось (вертикальная по умолчанию)

Свойства контейнера

.container {
    display: flex;
    flex-direction: row; /* или column, row-reverse, column-reverse */
    flex-wrap: nowrap; /* или wrap, wrap-reverse */
    justify-content: flex-start; /* или center, flex-end, space-between, space-around */
    align-items: stretch; /* или flex-start, center, flex-end, baseline */
    align-content: stretch; /* Для многострочных контейнеров */
}

CSS Grid

CSS Grid - это мощная система для создания двумерных макетов.

Основные понятия Grid

  • Grid container - родительский элемент с display: grid
  • Grid items - дочерние элементы контейнера
  • Grid lines - линии, разделяющие сетку
  • Grid tracks - пространство между двумя линиями (строки или столбцы)
  • Grid cells - отдельные ячейки сетки
  • Grid areas - прямоугольные области, состоящие из нескольких ячеек

Свойства контейнера

.container {
    display: grid;
    grid-template-columns: 100px 1fr 2fr; /* Определяет столбцы */
    grid-template-rows: 100px 200px; /* Определяет строки */
    grid-template-areas:
        "header header header"
        "sidebar content content"
        "footer footer footer";
    gap: 10px; /* Расстояние между элементами */
    justify-items: stretch; /* Выравнивание по горизонтали */
    align-items: stretch; /* Выравнивание по вертикали */
}

Адаптивный дизайн

Адаптивный дизайн позволяет создавать сайты, которые хорошо выглядят на всех устройствах.

Viewport

Метатег viewport необходим для корректного отображения на мобильных устройствах.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Медиа-запросы

Медиа-запросы позволяют применять стили в зависимости от характеристик устройства.

/* Для экранов шириной до 768px */
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

/* Для экранов шириной от 769px до 1024px */
@media (min-width: 769px) and (max-width: 1024px) {
    .container {
        width: 90%;
    }
}

/* Для экранов с ориентацией landscape */
@media (orientation: landscape) {
    .header {
        height: 80px;
    }
}

Относительные единицы измерения

.responsive {
    width: 100%; /* Проценты от родительского элемента */
    font-size: 2rem; /* Относительно корневого элемента */
    padding: 2em; /* Относительно размера шрифта элемента */
    margin: 5vw; /* 5% от ширины viewport */
    height: 50vh; /* 50% от высоты viewport */
}

Переходы (Transitions)

Переходы позволяют плавно изменять значения свойств в течение определенного времени.

Свойства переходов

.element {
    transition-property: all; /* Какие свойства анимировать */
    transition-duration: 0.3s; /* Длительность анимации */
    transition-timing-function: ease; /* Функция времени */
    transition-delay: 0s; /* Задержка перед началом */
    /* Сокращенная запись: */
    transition: all 0.3s ease 0s;
}

Функции времени (timing functions)

  • ease - медленно начинается, быстро в середине, медленно заканчивается
  • linear - постоянная скорость
  • ease-in - медленно начинается
  • ease-out - медленно заканчивается
  • ease-in-out - медленно начинается и заканчивается
  • cubic-bezier() - пользовательская функция

Анимации

Анимации позволяют создавать сложные последовательности изменений свойств.

Определение анимации

@keyframes pulse {
    0% {
        transform: scale(1);
    }
    50% {
        transform: scale(1.1);
    }
    100% {
        transform: scale(1);
    }
}

Применение анимации

.element {
    animation-name: pulse; /* Имя анимации */
    animation-duration: 2s; /* Длительность */
    animation-timing-function: ease-in-out; /* Функция времени */
    animation-delay: 0s; /* Задержка */
    animation-iteration-count: infinite; /* Количество повторений */
    animation-direction: normal; /* Направление */
    animation-fill-mode: none; /* Состояние до и после */
    animation-play-state: running; /* Состояние воспроизведения */
    /* Сокращенная запись: */
    animation: pulse 2s ease-in-out 0s infinite normal none running;
}

CSS переменные

CSS переменные (кастомные свойства) позволяют хранить значения для многократного использования.

Определение переменных

:root {
    --primary-color: #4361ee;
    --secondary-color: #3a0ca3;
    --font-size: 16px;
    --spacing: 1rem;
}

Использование переменных

.element {
    color: var(--primary-color);
    font-size: var(--font-size);
    margin: var(--spacing);
    /* Значение по умолчанию: */
    background-color: var(--bg-color, #f0f0f0);
}

Изменение переменных через JavaScript

// Получить значение переменной
const primaryColor = getComputedStyle(document.documentElement)
    .getPropertyValue('--primary-color');

// Установить новое значение
document.documentElement.style.setProperty('--primary-color', '#ff0000');
