Градиенты

Линейные, радиальные и конические градиенты

/* Линейный градиент */
.linear-gradient {
    background: linear-gradient(135deg, #667eea, #764ba2);
}

/* Радиальный градиент */
.radial-gradient {
    background: radial-gradient(circle, #667eea, #764ba2);
}

/* Конический градиент */
.conic-gradient {
    background: conic-gradient(from 0deg, #667eea, #764ba2, #f093fb, #f5576c);
}

Трансформации

Перемещение, масштабирование, поворот и наклон элементов

Наведи

}

.transform-demo:hover {
    /* Комбинированные трансформации */
    transform: translate(20px, -10px) scale(1.2) rotate(15deg) skew(5deg);

.translate {
    transform: translate(50px, 30px);

    transform: scale(1.5);

.skew {

}

.animation-demo.paused {

CSS Grid

Двумерные макеты с Grid

1
2
3
4
5
6
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 колонки равной ширины */
    grid-template-rows: auto; /* Автоматическая высота строк */
    gap: 10px; /* Расстояние между элементами */
    /* Альтернатива: grid-gap: 10px; */
}

.grid-item {
    /* Размещение элемента в сетке */
    grid-column: 1 / 3; /* От первой до третьей линии */
    grid-row: 1; /* Первая строка */
}

/* Определение областей сетки */
.grid-container {
    grid-template-areas:
        "header header header"
        "sidebar content content"
        "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }
