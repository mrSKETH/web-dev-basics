Введение в HTML

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц. Он определяет структуру и содержание веб-страницы с помощью различных элементов и тегов.

Важно!

HTML не является языком программирования. Это язык разметки, который сообщает браузеру, как отображать контент.

По этой причине вам не стоит говорить что вы "програмист" если изучили лишь HTML

    За счет HTML вы сможете указать какие объекты должны быть на странице: кнопки, текст, формы и так далее. HTML лишь позволит расставить все объекты и по началу они будут выглядеть не особо красиво.

    Тут вступает в силу CSS. Он даст вам возможность указать стили для любого HTML объекта. При помощи CSS и HTML вы уже сможете делать красивую разметку веб-сайтов. Ваши сайты будут ничем не хуже, чем полноценные веб проекты, что вы видите ежедневно на тысячах сайтов.

    В то же время, HTML и CSS, ни дает никакой функциональности. У вас будет сайт, он будет красивый, но без функций. По этой причине вам стоит ещё обязательно изучить JavaScript. Он позволит создавать пользовательские функции, что выполняются в браузере и для их выполнения не требуется перезагрузка страницы.

Создание html файлов

    При создании HTML файла обязательно указывать расширение файла, как: .html. Создав файл с таким расширением вы сможете открыть его через любой веб браузер. Некоторые разработчики указывают расширение .htm, что тоже является верным.

    Главная страница обычно всегда обозначается как index.html или index.htm

    Чтобы открыть HTML файл в браузере достаточно два раза нажать на него. После этого файл будет открыт в вашем браузере по умолчанию. Весь код из HTML файла будет обработан и показан на странице в браузере

Front-end и Back-end

    При создании веб-сайта всегда есть две стороны: Front-end и Back-end.

  • Front-end – это понятие, что описывает разработку внешнего вида сайта.
  • Back-end – это второе понятие, что описывает разработку серверной части сайта.

Важно!

В данном курсе мы будет рамастривать именно Front-end разработку

Основные понятия

  • Тег (Tag) - основной строительный блок HTML, заключенный в угловые скобки (< >)
  • Элемент (Element) - открывающий тег, содержимое и закрывающий тег
  • Атрибут (Attribute) - дополнительная информация об элементе

Структура HTML-документа

Каждая HTML-страница имеет базовую структуру, которая включает следующие элементы:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Заголовок страницы</title>
</head>
<body>
    <!-- Содержимое страницы -->
</body>
</html>

Объяснение структуры

Элемент Назначение
<!DOCTYPE html> Объявление типа документа (HTML5)
<html> Корневой элемент всей страницы
<head> Содержит метаинформацию о документе
<!----> Комментарий который будет виден лишь в коде но не конечному пользователю
<meta> Определяет метаданные (кодировка, viewport)
<title> Заголовок страницы (отображается во вкладке браузера)
<body> Содержит видимое содержимое страницы

Текстовые элементы

HTML предоставляет различные элементы для форматирования текста:

Заголовки

Заголовки определяются тегами <h1> до <h6>, где <h1> - самый важный заголовок.

Пример заголовков:

Заголовок 1 уровня

Заголовок 2 уровня

Заголовок 3 уровня

Заголовок 4 уровня

Заголовок 5 уровня
Заголовок 6 уровня

Тег абзаца и форматирование

<p>Это обычный параграф текста.</p>
<p>Это <strong>важный</strong> текст.</p>
<p>Это <em>выделенный</em> текст.</p>
<p>Это <mark>помеченный</mark> текст.</p>
<p>Это <small>маленький</small> текст.</p>
<p>Это <del>удаленный</del> текст.</p>
<p>Это <ins>вставленный</ins> текст.</p>

Объяснение структуры

Элемент Назначение
<p></p> Простой абзац
<strong></strong> Добавляет обёрнутому в него слову или фразе очень высокую важность
<em></em> По умолчанию выделяется курсивом
<mark></mark> Выделяет или подсвечивает важный фрагмент текста
<small></small> Используется для вывода текста мелким шрифтом
<del></del> Используется для отображения удалённого контента
<ins></ins> Используется для отображения добавленного контента

Списки

HTML поддерживает три типа списков:

Нумерованный список

<ol>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
    <li>Третий пункт</li>
</ol>
  1. Первый пункт
  2. Второй пункт
  3. Третий пункт

Маркированный список

<ul>
    <li>Пункт списка</li>
    <li>Еще один пункт</li>
    <li>И еще один</li>
</ul>
  • Пункт списка
  • Еще один пункт
  • И еще один

Список определений

<dl>
    <dt>HTML</dt>
    <dd>Язык разметки гипертекста</dd>
    <dt>CSS</dt>
    <dd>Каскадные таблицы стилей</dd>
</dl>
HTML
Язык разметки гипертекста
CSS
Каскадные таблицы стилей

Ссылки

Ссылки создаются с помощью тега <a> и атрибута href.

<!-- Внешняя ссылка -->
<a href="https://example.com">Посетите Example.com</a>

<!-- Внутренняя ссылка -->
<a href="about.html">О нас</a>

<!-- Ссылка на якорь -->
<a href="#section1">Перейти к разделу 1</a>

<!-- Ссылка с атрибутом target -->
<a href="https://example.com" target="_blank">Открыть в новой вкладке</a>

Атрибуты ссылок

Атрибут Назначение
href URL-адрес, на который ведет ссылка
target Определяет, где открыть ссылку (_blank, _self)
rel Отношение между текущим и связанным документом
download Указывает, что ссылка ведет на файл для скачивания

Изображения

Для вставки изображений используется тег <img>.

<img src="image.jpg" alt="Описание изображения" width="300" height="200">

Атрибуты изображений

Атрибут Назначение
src Путь к файлу изображения
alt Альтернативный текст (обязателен для доступности)
width и height Ширина и высота изображения в пикселях
title Всплывающая подсказка при наведении

Важно!

Всегда указывайте атрибут alt для изображений. Это важно для доступности и SEO.

Таблицы

Таблицы создаются с помощью тега <table> и содержат строки (<tr>), ячейки (<td>) и заголовки (<th>).

<table>
    <caption>Заголовок таблицы</caption>
    <thead>
        <tr>
            <th>Заголовок 1</th>
            <th>Заголовок 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Данные 1</td>
            <td>Данные 2</td>
        </tr>
    </tbody>
</table>
Заголовок таблицы
Заголовок 1 Заголовок 2
Данные 1 Данные 2

Элементы таблицы

Элемент Назначение
<table> Контейнер для всей таблицы
<caption> Заголовок таблицы
<thead> Заголовок таблицы (содержит <th>)
<tbody> Основное тело таблицы
<tfoot> Нижняя часть таблицы
<tr> Строка таблицы
<th> Ячейка заголовка
<td> Ячейка с данными

Формы

Формы используются для сбора данных от пользователей. Основной элемент - <form>.

<form action="/submit" method="post">
    <label for="name">Имя:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="message">Сообщение:</label>
    <textarea id="message" name="message"></textarea>
    
    <button type="submit">Отправить</button>
</form>

Основные элементы форм

Элемент Назначение
<input> Поле ввода (текст, email, пароль и т.д.)
<textarea> Многострочное текстовое поле
<select> Раскрывающийся список
<button> Кнопка
<label> Подпись для элемента формы
<fieldset> Группировка связанных элементов
<legend> Заголовок для fieldset

Семантические элементы HTML5

Семантические элементы четко описывают свое значение как для браузера, так и для разработчика.

<header>
    <h1>Заголовок сайта</h1>
    <nav>Навигация</nav>
</header>
<main>
    <article>
        <h2>Заголовок статьи</h2>
        <p>Содержимое статьи</p>
    </article>
    <aside>Боковая панель</aside>
</main>
<footer>Нижняя часть сайта</footer>

Основные семантические элементы

Элемент Назначение
<header> Вводная часть или навигация
<nav> Навигационные ссылки
<main> Основное содержимое документа
<article> Самостоятельный контент (статья, пост)
<section> Тематическая группа контента
<aside> Боковая панель или дополнительный контент
<footer> Нижняя часть документа или раздела

Преимущества семантических элементов

  • Улучшенная доступность для скринридеров
  • Лучшая SEO-оптимизация
  • Более понятная структура кода
  • Упрощенная стилизация с помощью CSS

Файловые эмелементы

В HTML есть два тега, что позволяют выполнить подключение CSS и JavaScript файлов.

<header>
    <link href="style.css" rel=stylesheet type="text/css">
    <script src="external.js"></script>

Атрибуты файлов

Элемент Назначение
<link> Определяет отношения между текущим документом и внешним ресурсом
<herf> Указывает ссылку или путь к подключаемому файлу
<rel> Описывает как связаны текущая страница и подключаемый ресурс
<type> Содержит тип связанного ресурса
Элемент Назначение
<script> Элемент кода, который позволяет совершать действия, включать анимацию и создавать другие эффекты на странице испульзуя JavaScript
<src> Указывает путь к файлу
